For 150 questions, the raw logic for each would be extremely long. Instead, here's a **compact cheat sheet** with the core algorithm/thought process for all questions.

# Level 1: Basic Logic

1. Even/Odd → `num % 2`
2. Positive/Negative/Zero → compare with 0
3. Largest of 2 → compare a and b
4. Largest of 3 → keep max variable
5. Smallest of 3 → keep min variable
6. Leap Year → `(year%400==0) || (year%4==0 && year%100!=0)`
7. Calculator → switch on operator
8. Swap with temp → temp=a; a=b; b=temp
9. Swap without temp → arithmetic/tuple swap
10. Multiplication Table → loop 1–10
11. Sum 1..N → accumulate sum
12. Factorial → multiply 1..N
13. Square → n*n
14. Cube → n*n*n
15. Divisible by 5 & 11 → both remainders 0

### Logic Mastery Rule

For every problem:

1. Understand input.
2. Understand output.
3. Solve manually on paper.
4. Identify repeating steps.
5. Convert repeating steps into loops/conditions.
6. Code it.
7. Dry-run with sample data.
